# Clipper — Chrome Extension

Chrome-расширение для извлечения контента веб-страниц и конвертации в Markdown. Отсекает рекламу, навигацию, футеры и прочий мусор, оставляя только полезное содержимое.

## Как это работает

Расширение использует [Mozilla Readability](https://github.com/mozilla/readability) — тот же алгоритм, что работает в режиме Reader View в Firefox. Он анализирует структуру страницы и определяет, что является основным контентом, а что — обвязкой.

Далее [Turndown](https://github.com/mixmark-io/turndown) конвертирует очищенный HTML в чистый Markdown.

Для x.com (Twitter) применяется отдельный экстрактор, который собирает текст твитов напрямую из DOM, поскольку стандартный алгоритм Readability не справляется со структурой твиттера.

## Возможности

- Извлечение контента любой веб-страницы в Markdown одним кликом
- Автоматическое удаление рекламы, навигации, футеров, сайдбаров
- Сохранение результата в `.md` файл или копирование в буфер обмена
- Поддержка x.com / Twitter — извлечение твитов, тредов, ленты
- Автоопределение языка в блоках кода
- GitHub Flavored Markdown — таблицы, зачёркивания, списки задач
- Quote-твиты извлекаются как цитаты
- Медиа в твитах сохраняются как markdown-ссылки на изображения

## Установка

1. Откройте `chrome://extensions` в браузере Chrome
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **Загрузить распакованное расширение** (Load unpacked)
4. Выберите папку с этим репозиторием (ту, куда его склонировали)

Расширение появится на панели инструментов браузера.

## Использование

1. Откройте любую веб-страницу или страницу x.com
2. Нажмите иконку **Clipper** на панели инструментов браузера
3. Расширение извлечёт контент и отобразит заголовок страницы
4. Выберите действие:
   - **Скачать .md** — сохранит Markdown файл с именем по заголовку страницы
   - **Копировать в буфер** — скопирует Markdown текст в буфер обмена

## Разрешения

Расширение запрашивает минимальный набор разрешений:

| Разрешение | Зачем |
|------------|-------|
| `activeTab` | Доступ к DOM активной вкладки для извлечения контента |
| `downloads` | Скачивание `.md` файла |
| `clipboardWrite` | Запись Markdown текста в буфер обмена |

## Поддерживаемые сайты

Работает на большинстве сайтов со статейным контентом: блоги, новостные сайты, документация, статьи на Medium, Dev.to, Habr и другие.

Отдельная поддержка x.com (Twitter):
- **Одиночный твит** — извлекает твит и тред (ответы автора)
- **Лента / профиль** — извлекает все видимые твиты
- **Quote-твиты** — отображаются как цитаты в Markdown
